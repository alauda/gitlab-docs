## v17.8.9

### Features and Enhancements

#### Template Capability Enhancements

With this release, the built-in `Quick Start` and `High-Availability` deployment templates now support Redis Sentinel mode. For configuration details, please refer to [Redis Credentials](../install/02_gitlab_credential.mdx).

## v17.8.4

### Features and Enhancements

#### Alauda Build of Gitlab

Includes Alauda Build of Gitlab v17.8.5 Addon that deploys Gitlab Community Edition.

Includes deployment using the following templates:

- Quick Start: A template for quickly creating a Gitlab instance, suitable for development and testing scenarios, not recommended for production environments.
- Production: A template for creating a production-ready Gitlab instance, recommended for production environments.


To know more about Gitlab v17.8 features, please visit [Gitlab v17.8 Release Blog](https://about.gitlab.com/releases/2025/01/16/gitlab-17-8-released/).
