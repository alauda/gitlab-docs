## v17.11.1

### Features and Enhancements

#### Alauda Build of Gitlab

The Alauda Build of Gitlab operator now integrates the community edition of Gitlab version 17.11.4. By this update, this operator enables users to leverage the latest features and improvements from the Gitlab community, supporting efficient software development workflows.

To know more about Gitlab v17.11 features, please visit [Gitlab v17.11 Release Blog](https://about.gitlab.com/releases/2025/04/17/gitlab-17-11-released/).
